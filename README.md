# SkyridingUI

## Overview
SkyridingUI is a World of Warcraft addon that restores the removed vigor-style UI for skyriding. 
It provides a clear visual representation of Skyward Ascent and Surge Forward charges (previously vigor).
The addon also includes additional modules for visual skyriding cues, such as Second Wind indicators, speed tracking, and buff animations.


## Features
- Restores the removed vigor-style UI for skyriding.
- Optional decor textures around the vigor bar.
- Second Wind visual cues.
- Speed Bar with optional text, units, and Thrill of the Skies marker.
- Progress bars for Skyward Ascent and Whirling Surge.
- Thrill of the Skies and Ground Skimming textures with pulse animation.
- Fully modular: enable only the pieces you want.

## Installation
1. Download the SkyridingUI addon.
2. Extract the contents into your World of Warcraft `Interface/AddOns` directory.

## Usage
- Once installed, log into World of Warcraft and enable SkyridingUI from the AddOns menu.
- Open options with `/skyui` or `/SkyridingUI`.
- You can also open the options, by clicking the SkyridingUI entry in the Addon Compartment at the Minimap.
- The vigor-style display activates automatically when you begin skyriding.
- Enable or disable any module to match your preferred setup.

## Modules
SkyridingUI is organized into focused modules under the `Modules` folder, each responsible for a skyriding mechanic or UI element.
- Vigor: core vigor bar and optional decor textures.
- Second Wind: optional Second Wind cues.
- Speed Bar: speed progress bar with optional numeric display and units.
- Skyward Ascent: progress bar for Skyward Ascent.
- Whirling Surge: progress bar for Whirling Surge.
- Thrill of the Skies: buff textures with configurable pulse animation.
- Ground Skimming: buff textures with configurable pulse animation.

## Options
- Only Vigor UI: hides all other modules while keeping the vigor UI active.
- UI placement: scale and position (X/Y) sliders, draggable option.
- Updates per second: animation update rate.
- Hide when grounded: automatically hides the UI when grounded.
- Vigor: enable/disable module/decor, and colors.
- Second Wind: enable/disable module, and colors.
- Speed Bar: enable, size, colors, preview, text display, units, and Thrill marker.
- Skyward Ascent and Whirling Surge: enable, size, colors, and preview.
- Thrill of the Skies and Ground Skimming: enable, colors, pulse type and speed, and preview.
